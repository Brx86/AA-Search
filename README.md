# AA-Search

Arch Linux Archive Search.

Powered by [Vue3](https://cn.vuejs.org/) and [Pico](https://picocss.com/).

## Build and run

```bash
git clone https://github.com/Brx86/AA-Search
cd AA-Search
yarn
yarn build
```
